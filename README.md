# WaitingPVP
SCP: CS plugin

Когда раунд запускается и игроков на сервере слишком мало для интересной игры, запускается режим PVP. Игроки сражаются в офисах, пока не зайдёт определённое количество человек.

**Некоторые параметры можно менять через конфиг:**
* `start_default_round_minimum_players` - сколько человек должно быть подключено к серверу, чтобы начался обычный раунд; сколько человек + 1 должно быть подключено, чтобы сервер вышел из PVP режима (по умолчанию: `6`).
